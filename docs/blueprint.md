# **App Name**: Blogwise

## Core Features:

- Article Listing and Pagination: Display the latest articles with pagination for easy navigation.
- Article Display: Show the article content, author information, and publication date.
- Category and Tag Organization: Enable readers to browse articles by categories and tags.
- Search Functionality: Implement a search bar for readers to find articles by keywords.
- Admin Authentication: Secure login page for admin and authors to access the dashboard.
- Article Management (CRUD): Create, read, update, and delete articles via the admin panel.
- Rich Text Editor with AI assistance: Provide a WYSIWYG editor for content creation. If the LLM tool detects that content could benefit from further detail, then recommend to the author to expand on a given topic.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of trust and sophistication.
- Background color: Light Gray (#F5F5F5), creating a clean, modern backdrop.
- Accent color: Teal (#009688) for interactive elements, drawing attention and creating contrast.
- Headline font: 'Space Grotesk', a geometric sans-serif, giving the website a modern techy look.
- Body font: 'Inter', which pairs well with 'Space Grotesk' as a body font for longer texts, making reading long articles easier.
- Simple, minimalist icons for categories and tags.
- Clean, grid-based layout that is responsive and mobile-first.
