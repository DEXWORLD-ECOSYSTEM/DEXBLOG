# Componente: SiteHeader

O `SiteHeader` é o componente de cabeçalho principal do **DEX BLOG**. Ele fica fixo no topo da página e fornece a identidade visual do site, a navegação principal, uma funcionalidade de busca e um design totalmente responsivo.

---

## Estrutura e Arquitetura

O componente foi refatorado para ser mais modular e seguir o princípio DRY (Don't Repeat Yourself). Ele é composto por sub-componentes reutilizáveis:

-   **`Logo`**: Um componente simples que renderiza o nome do site ("Blogwise") como um link para a página inicial. É usado em todos os layouts (desktop e mobile) para garantir consistência.
-   **`NavMenu`**: Um componente dinâmico que renderiza a lista de links de navegação a partir do array `navLinks`. Ele se adapta estilisticamente para ser exibido tanto na horizontal (desktop) quanto na vertical (painel mobile).
-   **`SiteHeader`**: O componente principal que orquestra a disposição do `Logo`, `NavMenu`, da barra de busca e do menu mobile (`Sheet`) com base no tamanho da tela.

---

## Código-Fonte Refatorado

Este é o código-fonte principal, localizado em `src/components/site-header.tsx`, agora utilizando uma abordagem de componentes.

```tsx
// src/components/site-header.tsx

'use client';

import Link from 'next/link';
import { Input } from '@/components/ui/input';
import { Button } from '@/components/ui/button';
import { Search, Menu } from 'lucide-react';
import {
  Sheet,
  SheetContent,
  SheetHeader,
  SheetTitle,
  SheetTrigger,
} from '@/components/ui/sheet';
import { useState } from 'react';
import { cn } from '@/lib/utils';

const navLinks = [
  { href: '/', label: 'Home' },
  { href: '/categories/technology', label: 'Technology' },
  { href: '/categories/productivity', label: 'Productivity' },
  { href: '/categories/lifestyle', label: 'Lifestyle' },
];

const Logo = () => (
  <Link href="/" className="font-headline text-2xl font-bold">
    Blogwise
  </Link>
);

const NavMenu = ({ onClick }: { onClick?: () => void }) => (
  <nav className={cn(
    "flex items-center space-x-6 text-sm font-medium",
    onClick && "flex-col space-y-4 space-x-0 mt-8"
  )}>
    {navLinks.map((link) => (
      <Link
        key={link.href}
        href={link.href}
        className={cn(
          "transition-colors hover:text-foreground/80 text-foreground/60",
          onClick && "text-lg"
        )}
        onClick={onClick}
      >
        {link.label}
      </Link>
    ))}
  </nav>
);

export function SiteHeader() {
  const [isSheetOpen, setSheetOpen] = useState(false);

  return (
    <header className="sticky top-0 z-50 w-full border-b bg-background/95 backdrop-blur supports-[backdrop-filter]:bg-background/60">
      <div className="container mx-auto flex h-16 items-center px-4">
        
        {/* Desktop Nav */}
        <div className="mr-4 hidden md:flex items-center">
          <Logo />
          <div className="ml-6">
            <NavMenu />
          </div>
        </div>

        {/* Mobile Nav */}
        <div className="md:hidden">
          <Sheet open={isSheetOpen} onOpenChange={setSheetOpen}>
            <SheetTrigger asChild>
              <Button variant="ghost" size="icon">
                <Menu className="h-5 w-5" />
              </Button>
            </SheetTrigger>
            <SheetContent side="left">
              <SheetHeader>
                <SheetTitle>
                  <Logo />
                </SheetTitle>
              </SheetHeader>
              <NavMenu onClick={() => setSheetOpen(false)} />
            </SheetContent>
          </Sheet>
        </div>

        <div className="flex flex-1 items-center justify-center md:justify-end">
           <div className="md:hidden">
              <Logo />
           </div>
           <div className="flex items-center justify-end md:flex-1 space-x-2 w-full max-w-sm">
              <div className="relative w-full">
                <Search className="absolute left-3 top-1/2 -translate-y-1/2 h-4 w-4 text-muted-foreground" />
                <Input
                  type="search"
                  placeholder="Search articles..."
                  className="pl-9"
                />
              </div>
            </div>
        </div>

      </div>
    </header>
  );
}
```

---

## Dependências e UI

-   **`lucide-react`**: Para os ícones de `Search` e `Menu`.
-   **`@/components/ui/*`**: Utiliza os componentes `Input`, `Button`, e `Sheet` para construir a interface.
-   **`@/lib/utils` (`cn`)**: Função utilitária para combinar classes do Tailwind CSS de forma condicional, essencial para a adaptação do `NavMenu`.
-   **`useState` (React)**: Para controlar o estado de abertura e fechamento do painel lateral (`Sheet`).

---

## Como Modificar e Manter

-   **Para adicionar/alterar links de navegação:** Graças à refatoração, basta modificar o array `navLinks` no topo do arquivo. O componente `NavMenu` garantirá que a mudança seja aplicada tanto no desktop quanto no mobile, sem a necessidade de alterações adicionais.
-   **Funcionalidade de Busca:** A lógica de busca continua pronta para ser implementada no `<Input />` principal.
-   **Estilo e Tema:** O estilo continua sendo gerenciado pelo `Tailwind CSS` e se adapta aos temas do `ThemeProvider`. A modularização facilita a aplicação de mudanças de estilo em `Logo` e `NavMenu` de forma isolada.
